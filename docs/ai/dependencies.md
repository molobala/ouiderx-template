# AI Dependency Index

This template uses OUIDERX.

Primary AI docs:

- In this repository, relative to the sample root: `../../docs/ai/index.md`
- In a standalone installed project: `node_modules/@ouidesigner/ouiderx/docs/ai/index.md`
- Public docs: `https://ouidesigner.orangemali.com/docs`

Project-local context:

- `.native-views.json`: available native host views and props.
- `.host-functions.json`: custom host functions available to Mevento.
- `.modules/`: installed OUIDERX modules.

If docs are missing from `node_modules`, inspect the package README and generated TypeScript declarations.
